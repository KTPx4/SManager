> [!NOTE]
> ### Control naming syntax for project
> - `Form` : frm[Name] , Ex: `frmMain`
> - `Panel` : pn[Name] , Ex: `pnTitle`
> - `Button` : btn[Name] 
> - `Picture Box` : ptb[Name]
> - `Label` : lb[Name]
> - `Datagrid Views` : dgv[Name]
> - `Check box` : ck[Name]
> - `Group box` : grb[Name]
> - `Combo box` : cbb[Name]
> - `Text box` :  txt[Name]
>
